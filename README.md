# TrueWorld

TrueWorld is an unofficial fan-game project heavily inspired by the mechanics and world of the "Pick Me Up!" manhwa. This is a non-commercial, educational project created for learning and portfolio purposes.

**Original manhwa (English):** https://mangabuddy.com/pick-me-up
